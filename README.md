## :toolbox: Getting Started

### :bangbang: Prerequisites

- Install Node Js in your computer<a href="https://nodejs.org/en"> Here</a>

### :running: Run Locally

Clone the project

```bash
https://github.com/vishalyd05/FoodVilla
```

Go to the project directory

```bash
cd FoodVilla
```

Install dependencies

```bash
npm i
```

Start the server

```bash
npm run dev
```
